Databinding - 2 way, 1 way, 1 time only

Using ng-model creates 2-way binding:

Double curly braces "{​{ }​}" create 1-way binding, that is what is listed here can't change the input, but it stays updated with the input field: {{name}}

One-time only looks like {​{ ::name }​}, and once initialized, it never update: {{::name}}
